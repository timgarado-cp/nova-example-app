## Installation

1. Clone the repository:

    ```bash
    https://github.com/timgarado-cp/nova-example-app.git
    ```

2. Navigate to the project directory:

    ```bash
    cd your-repo
    ```

3. Install dependencies:

    ```bash
    composer install
    ```

4. Publish the assets:

    ```bash
    php artisan nova:install
    ```

5. Run migrations:

    ```bash
    php artisan migrate
    ```
